# Focus trap
This directive will keep from moving to other focusable elements

## How to use
* `v-focus-trap` - set scope
```html
  <dialog v-focus-trap>
    ...
  </dialog>
```

## Used
* [SidePanel](/docs/organisms-sidepanel--terms-of-service)
