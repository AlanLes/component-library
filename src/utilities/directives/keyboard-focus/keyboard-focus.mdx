# Keyboard focus
After adding this directive focus state of element is visible only for keyboard navigation

## How to use
* `v-keyboard-focus` - set scope
* `.focus-is-hidden` - add this class to body
```html
  .focus-is-hidden {
    --border-radius-outline: 0;
    --focus-outer: none;
  }

  <body class="focus-is-hidden">
    ...
    <button v-keyboard-focus>
      ...
    </button>
  </body>  
```
