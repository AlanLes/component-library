# Click outside
After adding this directive you can call some function by clicking outside of element

## How to use
* `v-click-outside="functionName"` - bind function
* `v-click-outside:[boolean]="functionName"` - switch on/off directive
* `.stop` - must be added to outer element to stop propagation events
```html
  <UiButton @click.stop="toggleVisibility"></UiButton>
  <UiPopover 
    v-if="visibility"
    v-click-outside:[boolean]="toggleVisibility"
  >
    ...
  </UiPopover>
```

## Used
* [Datepicker](/docs/organisms-datepicker--common)
* [Dropdown](/docs/molecules-dropdown--with-content-slot)
