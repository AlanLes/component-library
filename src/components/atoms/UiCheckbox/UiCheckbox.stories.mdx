# Checkbox

This component should working like native checkbox in Vue.js. [Form Input Bindings - Vue.js](https://vuejs.org/v2/guide/forms.html#Checkbox)

<Canvas>
  <Story
    name="Common"
    args={{
      label: 'I read and accept Terms of Service and Privacy Policy',
      id: 'agreement'
    }}
  >
    {((args, {argTypes}) => ({
      props: Object.keys(argTypes),
      data() {
        return {
          checked: false
        }
      },
      components: {UiCheckbox},
      template: `<UiCheckbox v-model="checked" :id="id">{{label}}</UiCheckbox>`
    })).bind({})}
  </Story>
</Canvas>

This component support multiple choices checkboxes. And can pass `Object` as value.

<Canvas>
  <Story
    name="Multiple checkboxes"
  >
    {((args, {argTypes}) => ({
      components: {UiCheckbox},
      data() {
        return {
          selected: ['s_542']
        }
      },
      template: `<div style="display: grid">
        <UiCheckbox v-model="selected" value="s_542">I'm overweight or obese</UiCheckbox>
        <UiCheckbox v-model="selected" value="s_22">I smoke cigarettes</UiCheckbox>
        <UiCheckbox v-model="selected" :value="{id: 's_1190', priority: 'high'}">I've been recently injured</UiCheckbox>
      </div>`
    })).bind({})}
  </Story>
</Canvas>
