# Radio

This component should working like native radio in Vue.js. [Form Input Bindings - Vue.js](https://vuejs.org/v2/guide/forms.html#Radio)

<Canvas>
  <Story
    name="Common"

    }}
  >
    {((args, {argTypes}) => ({
      props: Object.keys(argTypes),
      components: {UiRadio},
      data(){
        return {
          picked: ''
        }
      },
      template: `<UiRadio
          v-model="picked"
          :id="id"
          :value="value"
        >{{ label }}</UiRadio>`
    })).bind({})}
  </Story>
