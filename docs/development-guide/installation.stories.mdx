## How to install
You can create a new project or integrate Symptom Checker UI Kit into an existing application. The installation process is mostly
the same for both cases. Choose the installation that best suits to your project.

```bash
npm i @symptom-checker/ui-kit
# or
pnpm add @symptom-checker/ui-kit
```

## Usage default styles
Symptom Checker UI Kit is a highly customizable UI components library. To customize components to your needs, you can use a lot of global
CSS variables and adjust styles to your requirements. For more details please <a href="/docs/getting-started-development-guide-css-customization--page">click here</a>.

We provide our default design if you want use it inside your project, you need to import them into the `main.js` file.

```js
import '@symptom-checker/ui-kit/src/styles/styles.scss'
```

## Usage components in SFC
You can directly import components from the Symptom Checker UI Kit and use them in a template.

### Script syntax

```html
<template>
  <UiComponentName>
</template>

<script>
import { UiComponentName } from '@symptom-checker/ui-kit'

export default {
  components: {
    UiComponentName
  }
}
</script>
```

### Script setup syntax

```html
<template>
  <UiComponentName/>
</template>

<script setup>
  import { UiComponentName } from '@symptom-checker/ui-kit'
</script>
```

### Internal components
Some components includes smaller internal components, you must import them using the full path, e.g `UiToggleButtonGroup` uses `UiToggleButton`

```html
<template>
  <UiToggleButtonGroup>
    <UiToggleButton/>
  </UiToggleButtonGroup>
</template>
<script setup>
  import { UiToggleButtonGroup } from '@symptom-checker/ui-kit'
  import UiToggleButton from '@symptom-checker/ui-kit/src/components/molecules/UiToggleButtonGroup/_internal/UiToggleButton.vue';
</script>
```
